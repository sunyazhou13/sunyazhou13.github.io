关于我

我们的一生像一个操作系统中的进程，日常活动是其中的线程，在时间片轮转调度中运行于队列 com.sunyazhou.self.queue.serial。我总结自己的学习经历为 一个孤独的线程——总是串行执行：英语、数学、各种……。为了记录那些容易遗忘的内容，我创建了这份 孙先生的工作笔记，它像一个开源工具箱，时常翻阅，也共享给需要的人。

我是 孙亚洲，来自祖国的北国冰城 哈尔滨。长期从事 iOS 应用开发，也搞过一段 macOS 开发，这是我所有技能中最擅长的领域。

📝 为什么写博客

写博客源于一段知识的存储演进：

栈 → 堆 → 引用 → 关键字索引 → 博客

起初大脑像空栈，不断压入知识直到栈溢出；改用堆存储后发现内存也 OOM；再转用引用标签存储，浏览器标签页密密麻麻像中毒了似的；最后只记关键字去搜索，却发现很多文章已被作者删除。加上自己也有技术积累需要分享，于是决定开博记录。

即便我的种种观点充满狭隘的理论和稚嫩的论证，也要知今是而昨非地做下去。否则会因虚度年华而悔恨，也会因碌碌无为而羞耻。

博客让我更沉静、不趋于浮躁地专注于自己的领域。偶尔分享技术和生活，不对品质苛求——毕竟也要花时间在工作上好好生活下去。
