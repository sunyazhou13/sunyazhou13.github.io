学习macOS开发的路线

开发总结

  1. 学习完所有__raywenderlich__的__macOS__开发教程
  2. __Cocoa Programming for OS X (5th Edition)__必看(英文版没有中文)
  3. __[Advanced.Mac.OS.X.Programming(3rd.2011)].Mark.Dalrymple.文字版__必看

以上这些必须看一遍 包括那两本epub的书籍 虽然都是英文的(我本人英语应该不算太好但也不差)看这个很easy

博客 > 老谭笔记 这个大牛是原新浪网盘mac端的技术开发他的每一篇文章我都看过并实践过，大家如果学习macOS开发我建议先认真扫一遍他的博客

喵神 巧神 这些博客都是每个iOS&macOS开发必看的 剩下的就是一堆博客了太多 我就不列了 大家github搜一下

欢迎大家提出问题 一起交流和学习，本人开发年头也不算太短了，以前搞不明白博客咋玩的，2017年开始决定先订个小目标 把博客搞好欢迎各位批评指教，鄙人定当咨诹善道察纳雅言，多谢！
